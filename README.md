# Actividad-de-evidencia-1_U2-Abstract-Factory
Patrones de diseño de software - Abstract Factory

Es un patrón de diseño creacional que proporciona una interfaz para crear familias de objetos relacionados o dependientes sin especificar sus clases concretas. Permite desacoplar el código cliente de las implementaciones específicas, facilitando el cambio de familias de productos y garantizando la compatibilidad entre ellos.
